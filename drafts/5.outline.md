### 4. Technical Mitigation Strategies
*   **Pattern-Based Scrubbing:** Implementing local regex-based anonymizers to detect and remove names and IDs before egress.
*   **Zero-Data-Retention (ZDR) Policies:** Selecting API endpoints that guarantee no data is stored for model training.
*   **The "Local-First" Architecture:** Moving sensitive processing to local environments before interacting with cloud LLMs.
