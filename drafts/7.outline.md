
# Breaking the AI Chatbox: Berkeley Students Build Autonomous Agents  

## 1. The Challenge of Dynamic Selectors in Modern Web Apps  
- **Problem**: Standard XPath/CSS selectors fail unpredictably in React/NextJS SPAs due to dynamically generated element IDs.  
- **Solution Approach**: Students pivoted to visual screenshots and "relative grounding" algorithms to identify UI elements.  
- **Impact**: Reduced selector fragility by focusing on visual landmarks rather than rigid DOM paths.  
- **Visual Needed**: Diagram comparing traditional selectors vs. visual grounding workflows.  

## 2. Evading Anti-Bot Security Systems  
- **Obstacles Encountered**: Headless browsers were instantly flagged by Cloudflare and automated Captcha systems.  
- **Bypass Strategies**:  
  - Stealth plugin profiles to mimic human behavior.  
  - Cookie injection from authenticated sessions.  
  - Residential proxy rotation to avoid IP reputation blacklisting.  
- **Outcomes**: Enabled uninterrupted automation for SaaS integrations and purchases.  
- **Visual Needed**: Table comparing effectiveness of anti-bot countermeasures (e.g., success rates, tool examples).  

## 3. State Machine Resilience via Checkpointing  
- **Risk**: Multi-step agent failures required complete restarts, wasting computation and tokens.  
- **Recovery Mechanism**: Implemented checkpoint recovery with state serialization to resume workflows mid-process.  
- **Efficiency Gains**: Reduced token costs by ~80% during iterative development.  
- **Visual Needed**: Flowchart showing state machine progression with checkpoint save/restore nodes.
