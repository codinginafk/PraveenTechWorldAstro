# Android Phone Not Charging: 10 Fixes (Try #3 First)  

## 1. Inspect and Replace Damaged Cables  
- **Description**: Confirm cable integrity using an inline USB-C power meter to measure wattage output.  
- **Key Steps**:  
  - Test cable with power meter; replace if wattage drops below expected levels (e.g., 15W → 2.5W).  
  - Try known working cables if no meter is available.
