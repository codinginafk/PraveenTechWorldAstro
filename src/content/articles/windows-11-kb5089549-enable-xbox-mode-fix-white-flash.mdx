1. Open Settings > Gaming > Game Mode
2. Toggle the new Xbox Mode option on

### Easy Step-by-Step Setup

1. **Check your version** (just press Win+R and type "winver")
   - You'll need build 22631+ for 23H2 or 26100+ for 24H2

2. **Install via [Windows Update](/articles/fix-windows-11-update-errors-2026-troubleshooting)**
   - Go to Settings → Update & Security → Check for updates

3. **Enable Xbox Mode**
   - Navigate to Settings → Gaming → Game Mode and toggle it on
   - Quick tip: Only enable it for games you actually play!

4. **Verify the fixes**
   - You should see no more white flashes in File Explorer
   - Enjoy smoother gameplay in supported games

### Troubleshooting Tips

If Xbox Mode won't switch on, try these steps:

2. Check if your DirectX version is up to date (needs DX12)
3. Install Xbox components through Optional Features
