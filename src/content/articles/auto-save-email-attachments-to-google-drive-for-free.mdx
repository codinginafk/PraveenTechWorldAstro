## Why This Matters

You know that feeling when your inbox explodes? Someone sends a critical invoice or a family photo, and it vanishes under three dozen other messages. Then you waste ten frantic minutes hunting for that one attachment when you need it right now. It’s a huge time suck.

Saving each file by hand is tedious and easy to forget. Important documents just slip through the cracks. What if every attachment could appear in a dedicated Google Drive folder, automatically? You'd save hours every month and never lose a file again.

## What You'll Learn

By the end of this guide, you'll be able to:
*   Connect Gmail to Google Drive using a free, built-in tool.
*   Create a simple, automated rule that runs without you touching anything.
*   Save attachments from all senders, or get specific with certain contacts.
*   Organize saved files into folders automatically by sender or subject.
*   Troubleshoot the most common setup hiccup in about 60 seconds.

## Step-by-Step Guide

### Setting Up Your First Auto-Save

The magic tool here is Google Apps Script. It sounds technical, but don't worry. It's just a small piece of pre-written code you'll copy and paste. Google provides it for free.

1.  Open a new browser tab and go to **script.google.com**. Sign in with the Google account you want to automate.
2.  Click the blue **+ New project** button. You'll see a blank coding screen.
3.  Look for the default text `function myFunction() { ... }` and delete all of it.
4.  Go to this page on my blog and copy the entire script. Paste it into the empty space in your new project. Don't worry, I'll explain what it does below.
5.  Click the disk icon to **Save project**. Give it a clear name like "Gmail to Drive Auto-Save."

### Granting Permissions

The script needs permission to access your email and drive. This is a one-time step.

1.  Click the **Run** button (the play icon) at the top.
2.  A box will pop up saying "Authorization required." Click **Review permissions**.
3.  Choose your Google account.
4.  You'll see a warning that "Google hasn't verified this app." This is normal because it's your own script. Click **Advanced**, then **Go to [your project name] (unsafe)**.
5.  Click **Allow** on the final prompt.

That's it. The script will run once, and you'll see "Execution completed" in the log at the bottom.

### Turning on the Auto-Saver

Now, let's make it run automatically. You don't want to press "Run" every day.

1.  In your script editor, click the **Triggers** icon (it looks like an alarm clock) in the left sidebar.
2.  Click the blue **+ Add Trigger** button in the bottom right.
3.  Set it up like this:
    *   Choose which function to run: `saveAttachmentToDrive`
    *   Which runs at deployment: `Head`
    *   Select event source: `Time-driven`
    *   Select type of time-based trigger: `Hour timer`
    *   Select the interval: `Every hour`
4.  Click **Save**.

From now on, every hour, the script will scan your inbox for new emails with attachments. Any it finds will be saved to a new folder called "Email Attachments" in your main Google Drive.

## Tips for Best Results

1.  **Create a dedicated Drive folder first.** The script makes one called "Email Attachments," but you can change the code to save directly into a folder you've already made. Just replace the folder name in the script.
2.  **Use labels for smarter organization.** The basic script saves everything to one folder. Want to get fancy? You can edit the script to save files based on the email's subject line or sender into different subfolders automatically.
3.  **Test it with yourself.** Send yourself an email with a test attachment. Within the hour (or you can trigger it manually now), check your "Email Attachments" folder in Drive. Seeing that file appear is a great feeling.
4.  **Mind your Drive storage.** Every attachment you save counts against your Google Drive storage limit. Keep an eye on it, especially if you get large video or design files regularly.
5.  **This only works for new emails.** The script won't go back and download attachments from old emails already in your inbox. It only processes new ones that arrive after you set it up.

## Frequently Asked Questions

**Is this really secure? Will someone else see my emails?**
The script runs entirely on Google's servers and uses your own account permissions. You're the only one who has access to the code and the resulting files. It's as secure as using any other Google Workspace feature.

**Do I need to pay for anything?**
Nope. Gmail, Google Drive, and Google Apps Script are all free for personal accounts. You're just using tools you already have in a smarter way.

**Will it save attachments from every single email, like newsletters?**
It will. That can clog up your Drive. If you want only specific people's emails, you can modify the script to filter by sender address. Or, as a simple fix, set up a Gmail filter to automatically label (and maybe archive) newsletters. Then tell the script to only save emails that *don't* have that label.
